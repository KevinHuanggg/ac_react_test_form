# S3-A3: My First React Form!

在完成基礎語法與功能學習後，接下來你要透過 React 實際完成第一個功能：表單！
在這份表單作業中會練習到組建 components (元件) 以及使用 usestate。

## 預期產出

1. 表單畫面，且能夠輸入名字與 Email
2. 按下 Submit 鍵後能夠在畫面成功印出欄位中的名字與 Email

## Getting Started

1. 確認電腦有安裝 npm 以及 node.js 環境
2. 將專案 clone 至本地
3. 安裝完畢後，使用終端機進入該資料夾輸入:

```
npm install
```

4. npm 安裝完畢後，輸入:

```
npm start
```

5. 進入網址:

```
http://localhost:3000
```

6. 取消掛載本地伺服器請於終端機輸入 ctrl + c

## Built With

- Node.js
- react 18.2
- Bootstrap 5.3
